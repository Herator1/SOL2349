# Combat Round

Each combat round represents a short period of intense activity aboard the ship.

A round usually includes several **ship actions**, each of which should involve at least one player roll.

Typical actions include:

- Targeting and sensor operations
- Maneuvering
- Weapon fire
- Point defense interception
- Engineering support

Not every crew role must act every round. The importance of different roles depends on the range and situation.

During Narrative Space Combat, the crew operates the ship through **Crew Actions**.  
Each action represents a crew member actively controlling a ship system.
# Crew Actions
Each action requires a **skill roll** using the character system.

```
d20 + Attribute + Skill
```

The GM sets a difficulty depending on the situation.

Typical values:

|Difficulty|Target|
|---|---|
|Easy|10|
|Standard|12|
|Hard|15|

---

# Results

Crew actions have three possible outcomes.

|   |   |
|---|---|
|Result|Effect|
|Success|Action works and provides the listed bonus|
|Critical Success|Natural 20. Bonus effect is stronger|
|Critical Failure|Natural 1. Action fails and creates a penalty|

# Gunnery Actions

**Primary Skill:** Gunnery

Direct fire weapons require attack rolls.

```
d20 + Gunnery + Armament
```

Compared against the target’s **Defense**.

|   |   |   |   |   |
|---|---|---|---|---|
|Action|Roll|Success|Critical Success|Critical Failure|
|Fire Railgun|Gunnery|Weapon hits|+1 damage|Weapon misaligned, cannot fire next round|
|Fire Laser Array|Gunnery|Weapon hits|+1 damage|Weapon overheats|

# Unmanned Crew Stations

If a crew station is **not occupied**, the ship cannot perform the corresponding actions.

|   |   |
|---|---|
|Missing Station|Effect|
|No Pilot|Ship cannot maneuver or change range|
|No Sensors|No Target Lock or electronic warfare|
|No Gunnery|Direct fire weapons cannot be used|
|No Engineer|No system boosts or repairs|
|No Command|Crew actions cannot receive coordination bonuses|

Crew members may move between stations between combat rounds if the ship layout allows it.
